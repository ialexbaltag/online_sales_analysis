# online_sales_analysis

Acesta este un mini proiect de inventariere a unor produse destinate vanzarilor.
Proiectul contine fisierile main.py, in care se instantiaza diverse obiecte, 
fisierul product.py, cu clasa Product in care sunt setate carcteristicile si informatiile despre produse,
fisierul product_manager, cu clasa ProductManager in care se regaseste lista de produse si informatiile despre aceasta.
